## Dataset Boundaries

- Open-SWE `org_id` means GitHub owner or owner-like label.
- Open-SWE real `user_id` truth is unavailable; report user-level results as N/A.
- Dataset B and T3 are trace-grounded semi-synthetic mechanism benchmarks.
- Synthetic A is controlled evidence only.
- tau-bench historical is an outdated historical trajectory supplement, not a current leaderboard
  reproduction.
- Cache-like and repaired timestamp fields are provider-view approximations.
